# My First Rhythm

**Level:** Beginner  
**Grade Range:** Grades 3–5 (ages 8–11)  
**Duration:** 45 minutes  
**Music Concept:** Note values (whole, half, quarter, eighth), beat, rhythm  
**CS Concept:** Loops — repeating a set of instructions  

---

## Overview

Students explore note values and rhythm by building a repeating drum
pattern using the **Repeat** block and the **Rhythm Maker** widget.
They discover that a loop in programming is like a repeating musical
phrase — the same instructions play over and over.

---

## Learning Objectives

By the end of this lesson, students will be able to:

- **Music:** Identify whole, half, quarter, and eighth note values
- **Music:** Clap and program a simple 4-beat rhythm pattern
- **CS:** Explain what a loop is and why loops are useful
- **CS:** Use the Repeat block to make a rhythm loop in Music Blocks

---

## Prerequisite Knowledge

- Lesson 1 (Hello, Pitch!) or basic familiarity with Music Blocks canvas
- Ability to count to 4 steadily

---

## Materials

- A device with a web browser and speakers/headphones
- [Music Blocks](https://musicblocks.sugarlabs.org)
- Blocks used: **Start**, **Repeat** (Flow palette), **Note Value**, **Drum** (Rhythm palette)
- Widget used: **Rhythm Maker** (accessible from the Rhythm palette)

---

## Activity

### Step 1 — Clap a rhythm (5 min)

1. Teacher claps a simple 4-beat pattern: clap–clap–rest–clap.
2. Students echo-clap.
3. Ask: *"If I wanted to play this pattern 4 times, how many claps is that total?"*
4. Introduce the idea: *"In music, we repeat patterns. In programming, we call that a loop."*

### Step 2 — Explore the Rhythm Maker widget (10 min)

1. Open Music Blocks and click the **Rhythm** palette.
2. Drag the **Rhythm Maker** block onto the canvas and click it to open the widget.
3. Show students the grid — each cell is a beat. Click cells to turn beats on/off.
4. Press **Play** inside the widget to hear the rhythm.
5. Ask students to create a pattern they like and remember it.

### Step 3 — Build a rhythm loop with blocks (15 min)

1. Close the Rhythm Maker widget.
2. Drag a **Start** block onto the canvas.
3. Inside Start, place a **Repeat** block (Flow palette) — set the repeat count to **4**.
4. Inside the Repeat block, add two **Note Value** blocks:
   - First: set to **1/4** (quarter note), with a **Drum** block inside (e.g. Kick Drum)
   - Second: set to **1/4**, with a **Drum** block inside (e.g. Snare Drum)
5. Click **Run** — the pattern plays 4 times.
6. Ask: *"What would happen if we changed the repeat count to 8?"*

### Step 4 — Modify and experiment (10 min)

1. Students change the note values (try 1/8 for faster beats, 1/2 for slower).
2. Students add a third drum sound to the pattern.
3. Challenge: create a pattern that sounds like a heartbeat (strong–weak–strong–weak).

### Step 5 — Reflect (5 min)

1. Ask: *"How is a loop in programming like a repeating musical phrase?"*
2. Ask: *"What would happen if music never repeated? Would it be easy to remember?"*

---

## Discussion Prompts

- What is the difference between a quarter note and an eighth note?
- Why do programmers use loops instead of writing the same code over and over?
- Can you think of a song that has a repeating drum pattern?
- What would change if you set the Repeat block to 1 instead of 4?

---

## Assessment Criteria

- [ ] Student can identify at least two different note values by name
- [ ] Student successfully builds a looping drum pattern using the Repeat block
- [ ] Student can explain what a loop does in their own words
- [ ] Student makes at least one intentional modification to their rhythm

---

## Extension Activities

- Combine a melody (Pitch blocks) with a drum loop in the same Start block
- Try nesting two Repeat blocks — what happens?
- Use the **Pitch-Time Matrix** widget to add pitched notes to the rhythm
- Challenge: program the rhythm of a well-known song (e.g. "We Will Rock You")
