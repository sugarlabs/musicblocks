Music Blocks - Meter Palette (Beginner) Test Suite

About This Test Suite

This test suite covers the beginner blocks in the Meter palette of Music Blocks. It tests the functionality of blocks related to meter, tempo, and beat counting.

Blocks Tested

  • Meter Block - Sets the musical meter (time signature)
  • Beat Count Block - Returns the current beat number
  • Master BPM Block - Sets the beats per minute tempo
  • Beat Factor Block - Returns the ratio of note value to meter
  • Every Beat Do Block - Executes actions on each beat

How to Use

1. Open this file in Music Blocks web application

2. The test will automatically load and you can run it

3. Check the console output for test results

4. Verify that all blocks are working as expected

Expected Results

When you run this test, you should see:

  • Beat count values from 1 to 4 (for 4/4 meter)
  • Beat factor calculations
  • Actions triggered on each beat
